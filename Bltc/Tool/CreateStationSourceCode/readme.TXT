Build Procedure :
	make
Run
1. 	cd run
	./demo.bat


Document :
1. After make (Build Procedure)
2.
	cd doc (after this, you are in tsm/doc directory)
	./demo.bat, then you can check the tis object (class TestItemArray), station object (class Station), Stations object (class StationArray) contents at each different time point of program run.

Environment:
  This code built on Windows with Cygwin. Required installed package is VC (cl.exe) and Cygwin with make, byacc and flex.
